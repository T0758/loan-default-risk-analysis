#Loan Default Risk Analysis

## Overview
This project analyzes borrower and loan data to identify key drivers of loan default risk.

## Tools Used
- SQL (CTEs, Joins, Aggregations)

## Key Findings
- Overall default rate: 24.29%
- Credit score 520-599 ad the highest default rate
- Higher debt-to-income (DTI) ratios are strongly associated with defaults
- Loans for weddings showed the highest default risk
- Age group 40-45 had the highest default risk
- Part-time employees showed higher default rates

## Risk Insights
The analysis highlights weaknesses in loan approval processes, particularly in
borrower affordability assessment and risk segmentation.

## Recommendations
- Introduce a minimum credit score threshold
- Implement DTI ratio limits
- Apply stricter checks for high-risk loan purposes
- Strengthen employment verification processes
- Develop a risk scoring model

## Outcome
Developed a rule-based risk scoring model to classify borrowers into low,
medium, and high-risk categories
